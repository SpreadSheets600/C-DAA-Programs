## Workflow

1. Create a dated folder like `16-04-2026/`.
2. Add a session `README.md` with exercise headings, algorithm, code, and output.
3. Put the matching `.cpp` or `.c` files in the same folder.
4. Run `npm run docs:sync` to regenerate the docs site and this README.
5. Run `npm run docs:build` when you want a production build.
